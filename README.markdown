G New Doc
===

Description
---
This is a google chrome extension, let you to create new Google Doc with only two clicks.

Avaliable types:

 * Document
 * Spreadsheet
 * Presentation
 * Drawing

Install
---

Install from [Chrome Web Store](https://chrome.google.com/extensions/detail/ebmobgdemdgppppmboipgkdfagfnleaj)
